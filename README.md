# CLIAtlas - A fast, searchable reference for everyday CLI commands.

![Git](https://img.shields.io/badge/Git-Commands-orange) ![npm](https://img.shields.io/badge/npm-Commands-red) ![pnpm](https://img.shields.io/badge/pnpm-Commands-yellow) ![Yarn](https://img.shields.io/badge/Yarn-Commands-blue) ![License](https://img.shields.io/badge/License-MIT-green)

The developer's map to command-line tools.

> CLIAtlas is a fast, searchable reference for the commands developers use every day. Whether you're working with **Git**, **npm**, **pnpm**, or **Yarn**, CLIAtlas helps you find the right command quickly-with explanation.

Stop googling the same command for the tenth time. Start shipping. CLIAtlas collects the commands developers actually use day-to-day, organized by what you're trying to do rather than buried in an alphabetical list.

## Commands

- 🔧 [Git](./docs/git.md) - setup, branching, syncing, undoing, stashing, history, tags
- 📦 [npm](./docs/npm.md)
- 📦 [pnpm](./docs/pnpm.md)
- 📦 [Yarn](./docs/yarn.md)
- ⚖️ [Package Manager Comparison](./docs/comparison.md) - side-by-side equivalents across npm/pnpm/Yarn
